리액트에 대해서

  • 리액트는 모바일 어필리케이션과 비슷한 ui를 제공하기 위한 웹 애플리케이션 제작도구
  • 리액트는 자바스크립트를 기반으로 reactjavascript와 reactDOM을 제작해서 제공
  • reactDOM은 reactjavascriplt 안에서 JSX라는 문법으로 html 구문을 작성할 수 있다.
  • 빌드된 리액트로 만든 어플은 html css js 파일로 되어 있다.
  • 리액트에서 함수와 클래스는 컴포넌트를 제작하기 위한 객체이고 컴포넌트(Component)는 ui를 구성하는 한 부분이다.(버튼, 내비게이션, 폼)

리액트의 구동 원리

  • 리액트는 root component가 여러 컴포넌트를 호출하여 reactDOM을 구성하고 화면에 render할 수 있는 htmlDOM을 만들고 출력한다. ReactDOM.render()
  • 이 때 component에게 초기값을 매개변수 props로 줄 수 있다.
  • props는 수정 불가한 초기값으로 컴퍼넌트가 렌더링할 때만 호출되길 기대한다.
  • props로 화면에 출력된 값은 바뀌지 않는다.
